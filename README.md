# 🚀 Network Intrusion Dashboard

## 📌 Overview

This project is a **Network Intrusion Detection Dashboard** that visualizes network activity and highlights potential security threats. It provides an interactive interface for analyzing data and identifying suspicious patterns.


## 🎯 Objective

To develop a user-friendly dashboard that helps in understanding network behavior and detecting possible intrusions through data visualization techniques.


## ⚙️ Features

* 📊 Interactive data visualizations
* 🚨 Detection of anomalous network activity
* 📈 Charts for traffic and threat analysis
* 🖥️ Clean and responsive UI
* ⚡ Fast performance using Vite


## 🧩 Project Structure

* **public/** – Static assets
* **src/** – Core application logic and components
* **index.html** – Entry HTML file
* **package.json** – Project dependencies
* **vite.config.js** – Vite configuration

## 🛠️ Technologies Used

* React
* Vite
* JavaScript (ES6)
* CSS

## 🚀 How to Run

```bash
npm install
npm run dev
```
